# FitSense AI Database Project

This repository contains SQL scripts and backup data used for the FitSense AI database project.

## Project Files

- [fitsense_ai_data_backup.sql](fitsense_ai_data_backup.sql) - Database backup dump.
- [import_csv.sql](import_csv.sql) - SQL script for importing CSV data.
- [mysql.sql](mysql.sql) - Core MySQL setup and schema script.
- [queries.sql](queries.sql) - Query collection for analysis and reporting.
- [DMA Mysql Implmentation.pdf](DMA%20Mysql%20Implmentation.pdf) - Project implementation document.

## Quick Start

1. Open [mysql.sql](mysql.sql) to create the database objects.
2. Use [import_csv.sql](import_csv.sql) to load CSV source data.
3. Run queries from [queries.sql](queries.sql) for validation and analysis.
4. Optionally restore from [fitsense_ai_data_backup.sql](fitsense_ai_data_backup.sql) if needed.
